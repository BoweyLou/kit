# Archive

This directory carries historical source and migration evidence from the old
repositories that were folded into `kit`.

- `agent-workflow-kit/` is a read-only import of the former
  `Codex_CodeReview` / `agent-workflow-kit` checkout. It exists for provenance,
  backlog closeout, and migration diffs.

Normal workflow-source, installer, CLI, template, and docs changes belong in
the root `kit` repository, especially `workflows/`, `scripts/`, `templates/`,
and `docs/`.
