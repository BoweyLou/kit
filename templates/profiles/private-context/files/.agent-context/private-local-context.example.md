# Private Local Context Example

Use this example only for local, non-secret reminders that should stay on this
machine. The copied non-example file is ignored by default, but ignored does not
mean safe to store sensitive data.

Do not include secrets, tokens, cookies, passwords, private URLs, account
identifiers, customer data, personal messages, medical or financial data, or
proprietary snippets that should not leave the machine.

Do not paste copied private-local context into hosted models, browser tools,
GitHub comments, pull requests, issues, external tickets, or chat tools unless
you have reviewed and removed sensitive or machine-specific material.

## Safe To Record

- Local command preferences that do not reveal private paths or accounts.
- Non-sensitive setup reminders for this checkout.
- Temporary local caveats that would be confusing in shared repo docs.

## Example Notes

- Check the repo's `make workflow-help` output before assuming command names.
- Keep durable follow-up work in the repo backlog or approved task tracker.
- Keep credentials and account access details in an approved secret manager.
